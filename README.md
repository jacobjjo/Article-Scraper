Article Scraper is a Python application used to process Excel files, detect a format similar to that used in the RECORDS Excel file used for the SEZARC UDEL Capstone Team 2026. Using UnpaywallAPI, Article Scraper will make requests using the DOIs located in the DOI column of the spreadsheet, automatically downloading them as well as generating a new Excel file to track which articles could be successfully downloaded. Currently, the success rate is ~20%, as article availability is highly dependent on how it is published. Moving forward, the UI can be updated to allow more versatility regarding filtering, valid Excel formats, and diagnostics.
